INSTRUCCIONES DE USO

Requisitos previos
Para compilar y ejecutar el programa, sigue los siguientes pasos (de preferencia utilizar vscode con las dependencias necesarias): 
Asegúrate de tener Node.js instalado en tu computadora. Puedes descargarlo e instalarlo desde su sitio web oficial: https://nodejs.org/en/download/package-manager/current

Paso 1: Instalación de dependencias en el backend
Navega hasta el directorio del proyecto clonado usando el comando cd:

  cd Sources
  cd Backend

Instala las dependencias del proyecto usando el siguiente comando:

  npm install

Paso 2: Ejecución del backend
Para ejecutar el programa, usa el siguiente comando:

  npm start
  
Paso 3: Instalación de dependencias en el Frontend
Navega hasta el directorio del proyecto clonado usando el comando cd:

 	cd ..
  cd Frontend
  cd file-transfomer
  
Instala las dependencias del proyecto usando el siguiente comando:

  npm install

Paso 4: Ejecución del Frontend
Para ejecutar el programa, usa el siguiente comando:

  npm start
